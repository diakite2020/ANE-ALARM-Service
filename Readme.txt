ANE-ALARM-SERVICE-Notification is a sample for using AIR to set scheduled task (a service to push notification) on android.

AlarmServiceApp is Flex main project which using ANE to start and stop the alarm.

AlarmServiceLib is library project which is part of ANE and link to Android API

NotificationLib is android project which set a alarm and then using server to push notification.

Compile folder contains all resources for compiling ANE, and the command
